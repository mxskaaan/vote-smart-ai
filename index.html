How it Works

  1. Register for Voter ID (Form 6)
  2. Verify name on Electoral Roll
  3. Show valid ID at polling booth
  4. Press EVM button to vote

Quick Tips

  • Carry original ID (Aadhaar/PAN)
  • Go early to avoid queues
  • Phones are not allowed inside
